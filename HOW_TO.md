## Про переключение языка

На GitHub **нельзя** сделать настоящий select/JS-переключатель в профиле.  
Работает так:

- сверху ссылки **English / Русский**
- клик **прыгает** к нужному блоку (якорь)
- внизу каждого блока — ссылка «Switch to …»

Это стандартный трюк для bilingual profile README.
